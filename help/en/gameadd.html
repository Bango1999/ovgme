Adding a config

To manage and install Mods for a game, a software or a destination folder, you have to add a config in the OvGME management list. To create a new configuration you can click on the "New Config" button or go to the "Config" menu then select "New...". A new dialog window will prompts you to specify informations about the new configuration to create.
Config Title
This is the main identifier for management list entry. This title can be what you want and can be modified later.

Root Path
This must be the path to the root destination folder. This is usually the game or software installation folder.

Mods Path
This must be the path to an existing Mods stock folder, this is where OvGME will seek for available Mods for this configuration.

Use custom backup folder
You can choose a custom backup folder instead of the default OvGME backup location.

Back Path
This must be the path to an existing folder, this folder will contain backup file tree for this configuration.
To activate a configuration you just have to select the desired entry in the management list combo box. Once you activated a configuration, the Mods stock folder and backup folder are analyzed to gather available Mods for installation or restoration.
