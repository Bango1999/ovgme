Supprimer un jeu

Vous pouvez supprimer l'entrée pour le jeu actuellement sélectionné. Pour supprimer le jeu actuellement sélectionné, allez dans le menu "Game", puis sélectionnez "Remove current..."

Si des données de sauvegarde existent pour ce jeu (i.e. si un ou plusieurs Mods est actuellement installé) OvGME va restaurer toutes les sauvegardes (désactiver tous les Mods) avant de supprimer la configuration du jeu. Le dossier stock de Mods n'est pas supprimé.
