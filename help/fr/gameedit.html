Editer un jeu

Vous pouvez éditer la configuration du jeu actuellement sélectionné. Pour éditer La configuration du jeu en cours, allez dans le menu "Game", puis sélectionnez "Edit current...". La boite de dialogue approprié apparaît.

Game Title
C'est l'identifiant principal pour le gestionnaire de jeux. Ce titre peut être ce que vous voulez.

Mods Path
Ce doit être un chemin vers un dossier existant de stock de Mods où OvGME ira regarder pour recueillir la liste des Mods disponibles pour ce jeu.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé. Ce dossier contiendra les fichiers sauvegardés pour ce jeu.

Back Path
Ce doit être le chemin vers un dossier existant où OvGME sauvegardera les fichiers originaux du jeu.

Note: Vous ne pouvez pas changer le dossier racine pour un jeu existant dans le gestionnaire de jeu, étant donné que le chemin vers le dossier racine du jeu doit être unique (pour éviter le chevauchement des Mods et des sauvegardes, vous ne pouvez pas créer plus entrée pour le même jeux dans le gestionnaire de jeux) et est l'identifiant principal de configuration de jeu. Si vous avez créé un jeu avec le mauvais chemin racine du jeu, vous devez le supprimer et réer une nouvelle entrée.
