Editer une config

Pour éditer la configuration actuellement activée, allez dans le menu "Config", puis sélectionnez "Edit current...". Une nouvelle fenetre de dialogue vous invite alors à spécifier les informations pour modifier la configuration actuelle.

Config Title
C'est le titre qui identifie la configuration dans la liste de gestion. Ce titre peut être ce que vous souhaitez et peut être modifié plus tard.

Mods Path
Ce doit être le chemin vers un dossier existant de stock de Mods, c'est là que OvGME recherchera la liste des Mods disponibles pour cette configuration.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé au lieu de l'emplacement de sauvegarde par defaut d'OvGME.

Back Path
Ce doit être le chemin vers un dossier existant, ce dossier contiendra l'arborescence des fichiers de sauvegarde pour cette configuration.
Note: Vous ne pouvez pas changer le chemin de destination racine (root path) pour une configuration existante. Si vous avez créé une configuration avec le mauvais chemin de dossier racine (root path), vous devez supprimer cette configuration et en créer une nouvelle.
