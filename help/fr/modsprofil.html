Profile de Mods

Le Profile de Mods vous permet de sauvegarder l'état actuel d'activation et et de désactivation des Mods, ceci pour l'appliquer ultérieurement. Il y'a un profile de Mods disponible par configuration de jeu.

Sauvegarder un profile de Mods

Pour sauvegarder l'état actuel des Mods en tant que profile, allez dans le menu "Mods", sélectionnez "Profile" puis "Save". La liste de Mods va être analysé pour sauvegarder l'état de chaque Mod, puis être enregistré.

Appliquer un profile de Mods

Pour appliquer un profile de Mods précédemment sauvegardé, aller dans le menu "Mods", sélectionnez "Profile" puis "Apply...". Chaque Mod sera activé ou désactivé selon le profile précédemment sauvegardé.
