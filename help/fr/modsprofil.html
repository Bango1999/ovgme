Profile de mods

Le Profile de mods vous permet de sauvegarder l'état actuel d'activation et et de désactivation des mods, ceci pour l'appliquer ultérieurement. Il y'a un profile de mods disponible par configuration.

Sauvegarder un profile de mods

Pour sauvegarder l'état actuel des mods en tant que profile, allez dans le menu "Mods", sélectionnez "Profile" puis "Save". La liste des mods va être analysé pour sauvegarder l'état de chaque mod, puis être enregistré.

Appliquer un profile de mods

Pour appliquer un profile de mods précédemment sauvegardé, aller dans le menu "Mods", sélectionnez "Profile" puis "Apply...". Chaque mod sera activé ou désactivé selon le profile précédemment sauvegardé.
