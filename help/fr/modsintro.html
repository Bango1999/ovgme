Introduction

Une fois que vous avez activé un jeu, le dossier stock de Mods pour ce jeu est analysé pour recueillir les Mods disponibles, aussi, les données de sauvegarde sont vérifié pour montrer les Mods que vous pouvez désactiver. Les Mods et données de sauvegarde disponibles sont énumérés dans la liste principale.

La liste de Mods

La liste de Mods vous montre les Mods disponibles et les sauvegarde pour le jeu actuellement activé. L'icone de droite vous montre le statu actuel du Mod.
: Indique un Directory-Mod disponible (non installé).

: Indique un Mod-Archive (Zip) disponible (non installé) .

: Indique un Mod installé (activé).
