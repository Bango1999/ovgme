Ajouter une config

Pour gérer et installer des Mods pour un jeu, un logiciel ou un dossier de destination, vous devez créer une nouvelle configuration dans la list de gestion d'OvGME. Pour créer une nouvelle configuration, vous pouvez cliquer sur le bouton "New Config" ou aller dans le menu "Config", puis sélectionner "New...". Une nouvelle fenetre de dialogue vous invite alors à spécifier les informations pour la nouvelle configuration à créer.
Config Title
C'est le titre qui identifie la configuration dans la liste de gestion. Ce titre peut être ce que vous souhaitez et peut être modifié plus tard.

Root Path
Ce doit être le chemin vers le dossier destination racine. Il s'agit générallement du dossier d'installation du jeu ou du logiciel.

Mods Path
Ce doit être le chemin vers un dossier existant de stock de Mods, c'est là que OvGME recherchera la liste des Mods disponibles pour cette configuration.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé au lieu de l'emplacement de sauvegarde par defaut d'OvGME.

Back Path
Ce doit être le chemin vers un dossier existant, ce dossier contiendra l'arborescence des fichiers de sauvegarde pour cette configuration.
Pour activer une configuration, vous devez simplement sélectionner l'entrée désirée dans la liste déroulante du gestionnaire. Une fois la configuration activée, le dossier stock de Mods et le dossier de sauvegarde sont analysés pour recueillir les Mods disponibles pour l'installation ou la restauration.
