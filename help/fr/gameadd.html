Ajouter un jeu

Pour gérer les Mods pour un jeu, vous devez ajouter un jeu dans la list de gestion d'OvGME. Pour ajouter un jeu, vous pouvez cliquer sur le bouton "Add Game" ou aller dans le menu "Game", puis sélectionner "Add...". La boite de dialogue appropriée vous invite à spécifier les informations de configuration du jeu à ajouter.
Game Title
C'est le titre qui identifie le jeu dans la liste de gestion. Ce titre peut être ce que vous souhaitez et peut être modifié plus tard.

Game Path
Ce doit être le chemin vers le dossier racine d'installation du jeu que vous voulez gérer. OvGME utilisera ce chemin pour identifier le jeu et construire les arborescences de fichier pour appliquer les Mods et créer les données de sauvegarde pour ce jeu.

Mods Path
Ce doit être le chemin vers un dossier existant de stock de Mods où OvGME regardera pour recueillir la liste des Mods disponibles pour ce jeu.

Use custom backup folder
Vous pouvez choisir un dossier de sauvegarde personnalisé. Ce dossier contiendra tous les fichiers de sauvegarde pour ce jeu.

Back Path
Ce doit être le chemin vers un dossier existant où OvGME sauvegardera les fichiers originaux du jeu.
Pour activer un jeu, vous devez simplement sélectionner l'entrée désirée dans la liste déroulante du gestionnaire de jeux. Une fois le jeu activé, la dossier de stock de Mods configuré est analysé pour recueillir les Mods disponibles, ainsi que les données de sauvegarde qui sont vérifiés pour afficher les Mods que vous pouvez désactiver.
