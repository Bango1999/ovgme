Importation de Mods

Vous pouvez importer un ou plusieurs Mod(s) pour le jeu actuellement sélectionné. Pour importer un ou plusieurs Mod(s), allez dans le menu "Mods" puis sélectionnez "Import Mod(s)..." ou cliquez sur le bouton "Import Mod(s)" de la fenêtre principale. Une fenêtre d'exploration vous invitera à sélectionner un ou plusieurs fichier .zip. Le processus d'importation va vérifier si les fichiers sélectionnés sont des Mod-Archives valides (voir aussi Structure de Mod), puis copier les Mods sélectionnés dans le dossier stock de Mods du jeu actuellement sélectionné.

Note: OvGME ne prend en charge que l'imporation de Mod-Archives zipés. Si votre Mod n'est pas un Mod-Archive valide (voir aussi Structure de Mod) vous devez extraire manuellement le sous-dossier du Mod dans le dossier stock de Mods. Si votre Mod est un simple sous-dossier, vous devez manuellement le copier dans dossier stock de Mods pour ce jeu.
