Mods profile

Mods Profile allow you to save the current enabled and disabled status for all current mods, so to be able apply it later. There is one Mods profile available per game configuration.

Saving mods profile

To save the current mods status as profile, go to the "Mods" menu, select "Profile" then "Save". The Mods list will be analysed to save the status of each mod then saved.

Applying a mods profile

To apply a previously saved Mods profile, go to the "Mods" menu, select "Profile" then "Apply...". Each Mod will be enabled or disabled according the previously saved profile.
